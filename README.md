# README

### 1. 브랜치만들기

```bash
$ git branch <branchname>
ex) $ git branch sm
```

### 2. 브랜치 전환

```bash
$ git checkout <branch>
ex) git checkout sm
```

### 3. 전환한 브랜치에서 작업

- 작업 후 add , commit 진행

### 4. 브랜치 merge

master branch로 이동

```bash
$ git checkout master
```

```bash
$ git merge <branchname>
ex) git merge sm
```
